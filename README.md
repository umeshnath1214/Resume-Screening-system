Description

A web-based application to automate resume screening using Natural Language Processing (NLP). The system analyzes uploaded resumes, extracts relevant skills, and compares them with job requirements to generate a match score. This helps recruiters quickly identify the most suitable candidates.
Installation

Follow these steps to run the project locally:

Clone the repository:
git clone https://github.com/your-username/resume-screening-system.git
Navigate to the project folder:
cd resume-screening-system
Install required dependencies:
pip install -r requirements.txt
Run the application:
python app.py
Open your browser and go to:
http://127.0.0.1:5000/
Usage
Register or login to the system.
Add a job with required skills and description.
Upload a candidate's resume (PDF, DOC, or DOCX format).
The system will:
Extract text from the resume
Analyze skills using NLP
Compare with job requirements
Generate a match score
View ranked candidates based on their match score in the results section.
